# 输出规则

## 1. 默认输出结构

每次完成任务后，默认输出：

1. 更新后段落；
2. 数据替换说明表；
3. Excel 文件链接。

## 2. 更新后段落要求

- 直接输出可复制到报告中的正文；
- 被替换部分用 Markdown 加粗；
- 未替换文字保持原样；
- 不额外插入过多解释；
- 如存在未替换数据，不要在正文中强行说明，除非影响理解。

## 3. 数据替换说明表要求

表头固定：

| 序号 | 原文数据 | 替换数据 | 数据来源 | 来源URL |
|---|---|---|---|---|

表格内容必须与 Excel 一致。

## 4. Excel 文件要求

Excel 文件名建议：

`数据替换来源表_YYYYMMDD.xlsx`

Sheet 名称：

`数据替换来源表`

字段顺序固定：

1. 序号
2. 原文数据
3. 替换数据
4. 数据来源
5. 来源URL

## 5. 未替换记录

如果未找到可核验数据，Excel 中仍记录一行：

- 原文数据：原始数据片段；
- 替换数据：未替换，暂未找到可核验的最新数据；
- 数据来源：-；
- 来源URL：-。
