# 输出规则

## 1. 默认输出结构

每次完成任务后，默认输出：

1. 更新后段落；
2. 数据替换说明表；
3. Excel 文件链接。

## 2. 更新后段落要求

- 直接输出可复制到报告中的正文；
- 不使用 Markdown 加粗；
- 不输出 `*` 或 `**`；
- 未替换文字保持原样；
- 不额外插入来源解释、口径解释或审阅痕迹；
- 不在正文中写“未替换”“暂未找到”“已知数据”等说明；
- 找到相近替代数据时，正文直接输出替代后的新句子。

## 3. 数据替换说明表要求

表头固定：

| 序号 | 原文数据 | 替换数据 | 数据来源 | 来源URL |
|---|---|---|---|---|

表格内容必须与 Excel 一致。

## 4. Excel 文件要求

Excel 文件名建议：

`数据替换来源表_YYYYMMDD.xlsx`

Sheet 名称：

`数据替换来源表`

字段顺序固定：

1. 序号
2. 原文数据
3. 替换数据
4. 数据来源
5. 来源URL

## 5. 未替换记录

只有在既未找到可核验同口径数据，也未找到可核验相近替代数据时，Excel 中仍记录一行：

- 原文数据：原始数据片段；
- 替换数据：未替换，暂未找到可核验的最新数据或相近替代数据；
- 数据来源：-；
- 来源URL：-。

该“未替换”记录只出现在表格和 Excel 中，不写入更新后段落正文。
