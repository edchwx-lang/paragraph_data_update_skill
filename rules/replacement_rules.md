# 数据替换规则

## 1. 总原则

替换遵循“准确优先、最小改动、来源可追溯”。

更新后的文本应尽量保留原文：

- 结构不变；
- 段落顺序不变；
- 逻辑关系不变；
- 不使用 Markdown 加粗；
- 正文不输出 `*` 或 `**`；
- 只替换旧时间、旧数值和必要的指标短语，不添加原文没有的解释结构。

## 2. 替换优先级

### A. 同口径精准替换

优先寻找同指标、同区域、同单位、同口径的2025年至今数据。

可直接替换的条件：

- 指标一致；
- 区域一致；
- 统计对象一致；
- 单位一致或可清晰换算；
- 来源可靠。

### B. 相关指标小句改写

如果找不到同口径数据，可以寻找与原句主题相关的最新权威数据，并只改写该句或分句。

适用场景：

- 原文写市场规模，但只能找到行业产值或营收；
- 原文写产量，但只能找到产能或出货量；
- 原文写企业数量，但只能找到规上企业数量或重点企业数量；
- 原文写全国数据，但只能找到权威机构公布的区域或细分领域数据，且能够支撑原句主题。

处理方式：

- 替换后的句子必须清楚说明新指标；
- 不得让读者误以为新指标与原指标完全一致；
- 只改写包含该数据的最小句子或分句；
- 保留原句中未受数据更新影响的后半句、判断句和趋势句；
- 不额外加入来源中出现但原文没有承载的新事实、新判断或新结构。

### C. 最新权威数据兜底

如果没有2025年至今数据，但有最新能找到的权威公开数据，可以使用该数据替换，但表述中必须体现最新时点。

示例：

- “截至2024年底……”
- “2024年……”

正文不得使用以下冗余前缀：

- “根据最新公开数据”
- “根据最新可核验口径”
- “公开资料显示”
- “已知数据”

### D. 相近例子替换

如果找不到同口径数据，但能找到意思相近、主题相关、来源可靠的新数据，直接用新数据替换该句。

要求：

- 正文只输出新句子；
- 不写“未替换”“暂未找到同口径数据”；
- 不在正文中解释为什么换口径；
- 口径变化写入数据替换说明表和 Excel。

### E. 完全无法替换

只有在既没有可靠同口径数据，也没有可靠相近替代例子时，才正文保留原文，Excel 中记录：

- 替换数据：未替换，暂未找到可核验的最新数据或相近替代数据
- 数据来源：-
- 来源URL：-

## 3. 正文标记规则

正文必须是可直接粘贴到报告中的成稿：

- 禁止 Markdown 加粗；
- 禁止输出 `*` 或 `**`；
- 禁止用“未替换”“暂未找到”“已知数据”等审阅痕迹污染正文；
- 禁止为了展示来源可靠性而在正文添加“根据最新公开数据”“公开资料显示”等前缀。

来源可靠性只体现在数据替换说明表和 Excel 中。

## 4. 最小改动规则

从小到大依次选择替换范围：

1. 只替换年份和数值；
2. 替换一个指标短语；
3. 改写一个分句；
4. 改写一个完整句子；
5. 不允许改写整段，除非用户明确要求。

替换时禁止：

- 找到新数据后同时保留旧数据；
- 写“由原先××提升至××”“较旧口径变化为××”等变化过程；
- 用来源材料里的其他数据扩写原句；
- 把原句没有的主体数量、企业数量、发展阶段、技术趋势等追加进去。

## 5. 不允许的替换

- 用全球数据替换中国数据，除非明确说明；
- 用企业营收替换行业规模但不说明口径变化；
- 用预测数据冒充实际数据；
- 用旧数据冒充2025年至今数据；
- 用来源不明的网页数据作为直接替换依据；
- 找到相近替代数据时，在正文写“未替换”“暂未找到同口径数据”。
