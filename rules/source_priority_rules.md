# 来源优先级规则

## 1. 来源优先级

检索最新数据时，按照以下顺序选择来源：

1. 国家部委、地方政府、统计局、官方统计公报、官方年鉴；
2. 行业协会、产业联盟、学会、权威智库；
3. 上市公司公告、年报、招股书、交易所文件；
4. 国际组织、咨询机构、科研机构报告；
5. 企业官网新闻稿、主流财经媒体；
6. 第三方数据库、转载站点，仅在无更高质量来源时使用。

## 2. 来源筛选要求

优先使用同时满足以下条件的来源：

- 有明确发布主体；
- 有明确发布日期或数据时点；
- 有具体指标名称；
- 有清晰统计范围；
- 有可访问 URL。

## 3. 不建议使用的来源

以下来源不得作为直接替换依据：

- 无发布主体的网页；
- 无日期的营销软文；
- 只摘录其他来源但没有原始出处的内容；
- 明显与原指标口径不同且无法解释的数据；
- 用户难以复核的二手内容。

## 4. 多来源冲突处理

如果多个来源数据不一致：

1. 优先采用政府或官方统计；
2. 其次采用行业协会或权威研究机构；
3. 如必须采用商业机构数据，应保留机构名称；
4. 不要混用不同来源的指标组成同一句结论；
5. 在替换数据中避免过度精确，可使用“约”“超过”“接近”等表述。
